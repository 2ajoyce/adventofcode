# Advent of Code 2025

This repository contains my solutions for Advent of Code 2025, implemented in Go.

## Challenge Types and Algorithms

| Day | Challenge Type                                   | Algorithm / Technique                                              |
| :-- | :----------------------------------------------- | :----------------------------------------------------------------- |
| 1   | [Simulation / Instruction Processing](1/main.go) | Modulo Arithmetic                                                  |
| 2   | [Interval / Range Processing](2/main.go)         | String & Numeric Pattern Detection                                 |
| 3   | [Sequence Analysis](3/main.go)                   | Two Pointers                                                       |
| 4   | [Grid Simulation](4/main.go)                     | Cellular Automata / Heatmap                                        |
| 5   | [Interval Queries & Union](5/main.go)            | Interval tree; union without-overlap + DFS counting                |
| 6   | [Expression Parsing](6/main.go)                  | String Manipulation                                                |
| 7   | [Grid with Beam Paths](7/main.go)                | Graph pathfinding / DP                                             |
| 8   | [Graph Connectivity / Clustering](8/main.go)     | Closest-pair sort + Disjoint Set Union (DSU)                       |
| 9   | [Computational Geometry](9/main.go)              | Brute-force rectangle search                                       |
| 10  | [State Space Search](10/main.go)                 | BFS / Integer-linear-system / Gaussian elimination                 |
| 11  | [Graph Pathfinding](11/main.go)                  | DFS + memoized DP (path counting)                                  |
| 12  | [Packing Problem](12/main.go)                    | Backtracking with pruning (precomputed placements, piece-ordering) |
